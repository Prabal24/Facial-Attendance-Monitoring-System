## Facial Attendance Monitoring System

A real-time facial recognition-based attendance system built using Python, OpenCV, and MySQL.

## 🚀 Features

 Real-time face detection
  Automatic attendance marking
  MySQL database integration
  Web dashboard

##  Tech Stack

Python (Flask)
 OpenCV
 MySQL
HTML, CSS, JavaScript
